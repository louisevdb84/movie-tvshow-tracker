## Set-up

Use the package manager [npm](https://docs.npmjs.com/cli/install) to install this project. Uses external urls to get images

### To clone first

```bash
git clone project
```

### To install dependencies

```bash
npm install
```

### To run
```bash
npm start
```
Then go to http://localhost:3000/
